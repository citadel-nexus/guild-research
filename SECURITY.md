# Security Policy

## Supported Versions

| Version | Supported |
|---------|----------|
| 0.x     | Yes       |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities to: **security@citadel-nexus.com**

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact assessment
- Your suggested fix (if any)

## Response SLA

We will acknowledge your report within **48 hours** and provide a
resolution timeline within 7 days.

## Scope

This policy covers:
- `guild-research` service code and NATS event handlers
- API endpoints and authentication flows
- Supabase query logic and data access patterns

Out of scope: third-party dependencies (report to their maintainers).

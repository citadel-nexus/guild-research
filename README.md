# guild-research

Simple service scaffold for the `guild-research` service.

- NATS prefix: citadel.research.*
- Port: 8200

Tech stack: TypeScript, Node.js, NATS JetStream, Supabase

Getting started:
1. Copy `.env.example` to `.env` and fill values.
2. `npm install`
3. `npm run build`
4. `npm start`
